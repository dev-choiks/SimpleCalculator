# (C# 코딩) 심플 사칙연산기 (SimpleCalculator)

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 사용자로부터 두 개의 숫자를 텍스트로 입력받아 계산이 가능한 정수형으로 변환한 뒤, 연산을 수행하고 그 결과를 화면에 출력하는 심플 사칙연산기 프로그램입니다.
- 사용한 플랫폼:
  - C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - Label, TextBox, Button
- 사용한 기술과 구현한 기능:
  - Visual Studio의 폼 디자이너를 이용하여 윈도우 계산기와 유사한 직관적인 UI 디자인 구성
  - int.Parse() 메서드를 활용하여 TextBox로 입력된 문자열(String) 데이터를 연산이 가능한 정수형(int) 타입으로 강제 변환 처리
  - C#의 사칙연산 기호 중 곱셈(*) 로직을 이용한 산술 연산 기능 구현
  - ToString() 메서드와 문자열 보간법($"")을 활용하여 산술 연산 결과를 다시 문자열로 되돌리고, 화면에 5 x 2 = 10과 같이 전체 수식과 최종 결과값으로 표시하는 로직 설계

- 과제 내용
  - Label(텍스트 표시), TextBox(수식 및 결과 표시), Button(숫자 패드 및 연산자) 컨트롤을 폼 화면에 적절히 배치합니다.
  - 숫자 버튼을 클릭했을 때 TextBox 영역에 해당 숫자가 텍스트 형태로 입력되도록 구현합니다.
  - 입력된 문자열을 int 형으로 바꾸어 연산을 수행한 뒤, 최종 결과를 다시 문자열로 변환하여 출력하는 데이터 흐름을 구현합니다.
  - 계산 내용 전체가 5 x 2 = 10과 같이 수식 형태로 모두 표시되는 방식과, 피연산자가 하나씩 표시되다가 최종 결과값만 표시되는 2가지 출력 방법을 동시에 구현합니다.

- 구현 내용과 기능 설명
  - Windows 폼 상단에 연산의 전체 과정을 기록할 보조 txtOutput 창을 두고, 그 아래에 현재 누른 숫자와 최종 결과값을 띄워줄 메인 txtInput 창을 배치하여 시각적인 데이터 흐름을 분리했습니다.
  - 0부터 9까지의 숫자 버튼들에 클릭 이벤트를 연결하고, 클릭할 때마다 sender를 이용해 해당 버튼의 텍스트 값을 가져와 메인 txtInput 창에 문자열 결합 방식으로 연속해서 이어 붙여지도록 코딩했습니다.
  - 사용자가 곱셈(*) 버튼을 클릭하면 메인 입력창의 텍스트를 int.Parse()를 이용해 숫자로 변환하여 전역 변수에 보관하고, 상단 보조 창에는 문자열 보간법을 사용해 첫 번째 피연산자와 곱셈 기호를 함께 출력하여 입력 내용을 표시했습니다.
  - 최종적으로 계산(=) 버튼 클릭 시 두 번째 피연산자까지 받아와 연산을 수행하며, 계산된 정수형 결과값은 ToString()을 통해 다시 문자로 가공하여 하단 txtInput 창에 결과값만 깔끔하게 띄우는 방식으로 입력(문자)에서 변환 및 처리(숫자)를 거쳐 출력(문자)되는 데이터 사이클을 성공적으로 완성했습니다.
